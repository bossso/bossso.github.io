CSS-Specificity คืออะไร?

CSS Specificity เป็นการเลือกใช้ CSS Selector ที่มีค่าความสำคัญมากกว่า มาใช้กับ Element บนหน้าเว็บไซต์

ทำให้ CSS Specificity เป็นตัวที่บ่งบอกค่าบางอย่าง ที่จะทำให้ browser ว่าต้องเลือกใช้ Style ตัวไหน

ภาษา CSS เหมือนภาษาโปรแกรมอื่น ๆ ที่อ่านโค้ดจาก บน – ล่าง ถ้าเขียน CSS Selector ซ้ำกัน 2 อัน อันบนจะถูกโดนทับ

และ อันล่างจะถูกนำไปใช้ นั่นแสดงว่า โค้ดจะนำส่วนด้านล่าง ซึ่งใหม่กว่านำไปใช้นั่นเอง

แต่ความพิเศษของ CSS คือ Selector ที่แตกต่างกัน จะมี ค่าความสำคัญ ที่ไม่เหมือนกัน ทำให้เมื่อ ค่าความสำคัญนี้ต่างกัน

ลำดับก่อนหลังจึงไม่สำคัญโดยทันที

วิธีคำนวณ ค่า CSS-Specificity

เราสามารถคิดค่าความสำคัญของ Selector ได้โดย

  1. การเขียน CSS แบบ internal style sheet มี ค่าเท่ากับ 100 level
  2. การเขียน CSS แบบ ใส่ ID มี ค่าเท่ากับ 5 level
  3. การเขียน CSS แบบ ใส่ Class มี ค่าเท่ากับ 2 level
  4. การเขียน CSS แบบ ใส่ Element มี ค่าเท่ากับ 1 level

สำหรับ !important ที่ไม่ได้อยู่ในข้อ 1 – 4 ด้านบน ถ้าใส่เข้าไปจะเป็นการบังคับใช้ CSS นั้นทันที โดยไม่ต้องทำการตรวจสอบ ค่าความสำคัญ

สรุปเรื่อง CSS SPECIFICITY

อย่าใช้ ID ถ้าไม่จำเป็นเพราะ มันเป็นการทำให้ เว็บเราโหลดหนักขึ้นและ ถ้าอาจทำให้ Selector โดนทับโดยไม่จำเป็น

แนะใช้ class ดีที่สุดเพราะจัดการได้ง่ายกว่ามาก
